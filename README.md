# ЭРКАФАРМ — Feed Scanner

Проверка наличия товара в YML-фидах аптек сети **ЭРКАФАРМ** (озерки, Столетов, Самсон Фарма и др.).

Отдельный инструмент: без amwine и прочих «онлайн» проектов.

## Возможности

- Вход через **Dashboard + TOTP**
- Список партнёров сохраняется в `DATA_DIR` (на Railway — Volume `/data`)
- **Кэш XML** на диске — повторный прогрев не обязателен после перезапуска
- Поиск по URL / slug / offer id
- Добавление партнёра через **+** (мастер-фид `feeds_list.json`)
- Ссылка «Ссылка на фид» в результатах
- **По всем брендам** — один поиск по всем фидам всех партнёров, выдача по сворачиваемым брендам

## Запуск локально

```bash
npm install
npm start
```

http://127.0.0.1:4173 → логин Dashboard → работа.

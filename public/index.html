Ниже — два разных режима поиска. Кэш обновляется только кнопкой «Обновить кэш» в шапке.

Раздел 1 · вся сеть

Поиск по всем брендам

Сканируются фиды всех партнёров сразу. Выбор site_id здесь не нужен — бренд не выбирается.

Раздел 2 · один бренд

Поиск по одному бренду

Сначала выберите партнёра — поиск и фильтры фидов относятся только к нему.

Мастер-фид выбранного бренда: —
В YML часто нет «названия» товара — только id, url, price. Поиск по URL / slug или по <offer id="...">.
